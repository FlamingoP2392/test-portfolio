About Me

ปัจจุบัน ผมเป็นนักศึกษาระดับประกาศนียบัตรวิชาชีพชั้นสูง ปีที่ 2 สาขาวิชาเทคโนโลยีสารสนเทศ วิทยาลัยชุมชนอุทัยธานีครับ

ตัวผมในตอนที่ยังเด็ก มีจุดเริ่มมาจากความสงสัย ว่าโปรแกรม แอปพลิเคชั่น รวมไปถึงเกมต่างๆ ในคอมพิวเตอร์และในสมาร์ทโฟน มันเกิดขึ้นมาได้อย่างไร มีวิธีการสร้างแบบไหน ทำให้ผมมีความสนใจ และเริ่มศึกษาเกี่ยวกับการเรียนในด้านคอมพิวเตอร์ครับ และด้วยเหตุนี้เอง จึงทำให้ผมได้รู้ ว่าเบื้องหน้าที่เป็นเทคโนโลยีสุดล้ำที่เราใช้ในชีวิตประจำวัน ในเบื้องหลังจะต้องมีขั้นตอนการศึกษา ค้นคว้า การลองผิลองถูกมากมาย กว่าจะได้มาซึ่งเทคโนโลยีสักอย่างหนึ่ง ซึ่งผมมองว่ากระบวนการเหล่านั้น เป็นสิ่งที่น่าหลงใหล ท้าทาย และช่วยฝึกฝนให้เราสามารถคิดวิเคราะห์ แก้ปัญหาต่างๆ ได้ นี่จึงเป็นเหตุผลหนึ่งที่ผมชอบในการเรียนเกี่ยวกับคอมพิวเตอร์ และทำให้ผมสร้างเว็บไซต์เล็กๆ แบบนี้ขึ้นมาได้ครับ

นอกจากผลงานทางทักษะวิชาชีพและทางวิชาการข้างต้น ผมยังมี Repository ของ Github ที่รวบรวมงานที่ผมได้ศึกษาเรียนรู้ด้วยตัวเองในการทำเว็บไซต์ด้วยครับ คลิก --> My GitHub
